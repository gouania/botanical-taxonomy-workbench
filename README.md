# Botanical Taxonomy Workbench 🌿🔬

The **Botanical Taxonomy Workbench** is the unified evolution of "The Biologist's Toolkit." It integrates several experimental AI-powered botanical tools into a single, interconnected workspace designed for systematic botany and herbarium workflows.

## 📖 Overview
This workbench moves away from isolated "apps" and toward a unified environment where taxonomic data is linked. Powered by **Gemini 3 Flash**, it uses search-grounded reasoning to assist with identification, morphological analysis, and historical research.

## ✨ Integrated Modules
- 🔑 **Identify**: A search-grounded multi-access key for plant families. Includes a "Suggest Next Feature" reasoning engine.
- 🔬 **Profiles**: Generate detailed diagnostic morphological profiles or compare up to three taxa side-by-side.
- 📜 **Authorities**: Biographical search for plant taxonomists. **New:** Taxa mentioned in an author's legacy now link directly to their morphological profiles.
- 📖 **Guide**: A new module for generating custom identification guides and dichotomous keys from raw text or AI synthesis.

## 🔗 New Interactivity
The workbench introduces deep linking between modules. For example:
- Looking up an author in **Authorities** allows you to click on their described taxa to instantly open a **Taxon Profile**.
- Viewing a **Taxon Profile** allows you to click on the author citation to view their biographical **Authority** profile.

## 🛠 Tech Stack
- **Frontend:** React, TypeScript, Tailwind CSS
- **AI/LLM:** Google Gemini 3 Flash (with Search Grounding)
- **Icons:** Lucide React
- **Deployment:** Google Cloud Run

## ⚠️ Disclaimer
This is an **experimental prototype**. All information is AI-generated and must be verified against primary taxonomic literature, authoritative floras, or herbarium specimens.
